# 暖宠归家

流浪动物救助与领养平台，使用 HBuilder X / uni-app 开发。

当前阶段：项目骨架与页面路由已创建。

## Tab 图标路径

请将 8 张底部导航图标放到 `static/tabbar/` 目录，并保持下面的文件名：

- `adopt.png`
- `adopt-active.png`
- `community.png`
- `community-active.png`
- `guide.png`
- `guide-active.png`
- `profile.png`
- `profile-active.png`

建议尺寸：81px * 81px，单张小于 40KB。
